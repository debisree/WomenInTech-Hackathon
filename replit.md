# WomenInTech Hackathon

## Overview
A web application starter project for the WomenInTech Hackathon. Currently set up as a Node.js/Express application serving static files.

## Project Architecture
- **Runtime**: Node.js 20
- **Framework**: Express.js
- **Entry point**: `index.js` - Express server on port 5000
- **Static files**: `public/` directory (HTML, CSS, JS)

## Project Structure
```
├── index.js          # Express server
├── public/
│   ├── index.html    # Main HTML page
│   └── style.css     # Styles
├── package.json      # Dependencies
└── .gitignore
```

## Running
The application starts via `node index.js` and serves on port 5000.
